# Instituto Tanto Mar — Site

Repositório de conteúdo e assets para o site do Instituto Tanto Mar, organização dedicada à restauração de recifes de coral através da tecnologia Bio-Rock.

## Estrutura

```
tanto-mar-site/
├── brand/
│   ├── logo/           # Logotipos e variações
│   ├── colors.md       # Paleta de cores
│   └── typography.md   # Tipografia
├── assets/
│   ├── photos/
│   │   ├── reefs/      # Fotos de recifes e Bio-Rock
│   │   ├── fieldwork/  # Trabalho de campo
│   │   └── education/  # Ações educativas
│   └── videos/         # Vídeos do projeto
├── content/
│   ├── about.md        # Sobre o instituto
│   ├── projects.md     # Projetos e iniciativas
│   ├── team.md         # Equipe
│   └── contact.md      # Contato
└── references/
    └── inspiration/    # Referências visuais e de design
```

## Sobre o Instituto

O Instituto Tanto Mar é uma associação civil sem fins lucrativos fundada em 2025, dedicada à conservação e restauração de ecossistemas marinhos no Brasil.

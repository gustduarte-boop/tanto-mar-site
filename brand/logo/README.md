# Logo do Instituto Tanto Mar

## ✅ Versão oficial — `tanto-mar-logo-oficial-hires.png`

Extraído do **Estatuto do Instituto Tanto Mar** (documento final, 2025) a 600dpi.

- Punho fechado erguido com **polegar visível na parte de cima**
- Ondas estilizadas saindo da base do punho
- Degradê de **azul oceano (topo) para verde-mar (base)**
- Tipografia "TANTO MAR" em cinza, peso regular, caixa alta, espaçamento amplo

## ❌ Versão antiga — `tanto-mar-logo-versao-antiga.png`

Versão anterior presente no TantoMar_R2.pdf.

- Fist mais estilizado/abstrato — **polegar em posição diferente**
- Traço mais fino, proporções distintas
- **Não usar** — foi substituída
